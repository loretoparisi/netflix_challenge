INTRODUCTION

This is where we'll store some of our blended predictions, as well as the
quiz RMSEs they achieved (added on in the same format as in the
"quiz_blend" folder). Additional details on each blend are below.

ADDITIONAL DETAILS
===========================================================================

BLEND_MAY_25_QRMSE_0.86810: These results came from quiz bin blending with
15 evenly-spaced bins (whose boundaries were set by how many movies each
user rated). Lambda was set to 0.0 since it's possible that 0.0014 is too
high for the small datasets involved in quiz bin blending (this also
noticeably improved the QRMSE). However, it's possible that lambda = 0.0
allowed for very highly correlated predictors that in turn led to
overfitting on the quiz set.

As a summary of all of the bins' blending coefficients, see below. Note
that many of these coefficients go all over the place, which could be an
issue, or just indicative of the small datasets (around 1/15 of the entire
dataset).
